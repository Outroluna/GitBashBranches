# **Bootstrap Ruby Gem** ![Static Badge](https://img.shields.io/badge/build-passing-green) ![Static Badge](https://img.shields.io/badge/gem-v4.1.1-blue)


[Bootstrap 4](https://bootstrap.com) gem for Ruby on Rails (Sprockets) and Hanami (formerly Lotus).

For Sass versions of Bootstrap 3 and 2 see [bootstrap-sass](https://bootstrap.com) instead.

## Installation

Please see the appropriate guide for you environment of choice:
- [Ruby on Rails 4+](https://bootstrap.com) or other Sprockets environment.
- [Other Ruby frameworks](https://bootstrap.com) not on Rails.

### a.Ruby on Rails
Add `bootstrap` to your Gemfile:
```
gem 'bootstrap', '~> 4.1.1'
```

Ensure that `sprockets-rails` is at least v2.3.2.

`bundle install` and restart your server to make the files available through the pipeline.
